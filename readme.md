## Setup

- Set up Laravel 5
- composer install
- php artisan key:generate

- Go to root
- ember new project_name --skit-git

- project_name/.ember-cli 
- "watcher": "polling"
- "liveReload": true

## Build

ember serve

bash *shell_file.sh* (e.g. ember_build.sh)
